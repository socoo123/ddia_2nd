## 笔记结构说明

每章笔记统一采用以下结构:

```
1. 📚 核心论文与参考文献 ← 放最前面，方便快速回查
2. 🗺️ 章节概览与 Mermaid 架构图
3. 📖 详细内容（按小节展开，约为原书 1/3 浓缩）
4. 💻 代码示例与最佳实践
5. 🎯 系统设计面试题（含思路分析）
6. 📝 本章要点总结
```

## 如何使用这份笔记

- **快速复习**: 看每章开头的架构图 + 末尾的要点总结
- **深入理解**: 按小节展开阅读详细内容
- **查找论文**: 每章开头的论文列表，附有中文资源链接
- **面试准备**: 直接看每章末尾的系统设计题
- **实践参考**: 查看代码示例与最佳实践部分
